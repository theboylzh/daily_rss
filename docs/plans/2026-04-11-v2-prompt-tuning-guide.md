## 阶段 1：概要生成提示词

**位置**: `_get_stage1_prompt()` (约第 50-92 行)

### 常见问题与修改建议

| 问题 | 症状 | 修改建议 | 修改位置 |
|------|------|----------|----------|
| JSON 解析失败 | AI 输出带 Markdown | 强化"纯 JSON"要求 | 提示词开头 |
| one_liner 过长 | 超过 30 字 | 添加字数限制和示例 | one_liner 字段说明 |
| keywords 太泛 | "发展""创新"等 | 要求"具体、有信息量" | keywords 字段说明 |
| briefing 某类为空 | 政治/经济等缺失 | 要求"每类至少 50 字" | briefing 说明 |

### 推荐 Prompt 模板

```python
【重要要求】
1. 必须输出纯 JSON，不要任何 Markdown 格式（如 ```json）
2. 不要任何额外说明文字
3. JSON 必须能被 Python json.loads() 直接解析

one_liner 字段要求:
- 严格限制 20-30 个汉字
- 高度抽象，理性简洁
- 示例："AI 行业震动：Sora 关停，巨头加速布局，油价影响消费"
- 错误示例："今天 AI 行业发生了很多事情，包括 Sora 关停和苹果布局 AI"（太长且具体）

keywords 字段要求:
- 3 个关键词，每个 2-4 字
- 必须具体、有信息量
- 正确示例：["Sora 关停", "油价上涨", "AI 手机"]
- 错误示例：["发展", "创新", "科技"]（太泛）
```

## 阶段 2：观点生成提示词

**位置**: `_get_stage2_prompt()` (约第 111-141 行)

### 常见问题与修改建议

| 问题 | 症状 | 修改建议 |
|------|------|----------|
| 观点空洞 | "AI 很重要" | 要求论述框架：现象→原因→影响 |
| 观点重复 | 3 个观点说同一件事 | 要求"独特视角、互不重复" |
| references 为空 | 缺少引用 | 要求"必须引用 2 篇相关文章" |

### 推荐 Prompt 模板

```python
perspectives 生成要求:
1. 生成恰好 3 个观点
2. 每个观点必须有独特视角，互不重复
3. 观点要鲜明，不要模棱两可

description 论述框架:
1. 描述现象/事实（50 字）
2. 分析背后的原因/趋势（100 字）
3. 指出影响/启示（50 字）
4. 总字数 200-300 字

references 引用要求:
- 从提供的新闻列表中选择 2 篇与观点最相关的文章
- 必须包含 title 和 url
```

## 阶段 3：深度分析提示词

**位置**: `_get_stage3_prompt()` (约第 143-176 行)

### 常见问题与修改建议

| 问题 | 症状 | 修改建议 |
|------|------|----------|
| facts 含主观 | 事实描述带观点 | 强调"客观、无评价" |
| viewpoint 模糊 | "有一定影响" | 要求"鲜明判断" |
| causes 浅层 | "因为各种原因" | 要求"主体 + 动机"结构 |
| prediction 含糊 | "可能会...也可能" | 要求"明确方向" |
| advice 空洞 | "注意风险" | 要求"具体可操作" |

### 推荐 Prompt 模板

```python
deep_analysis 字段定义（严格遵守）:

facts 客观事实:
- 只描述谁、何时、何地、做了什么
- 必须有明确的主体（如"国家发改委"）
- 必须有具体的时间或场合
- 不能包含主观评价（如"重大影响"）
- 示例："国家发改委于 2026 年 3 月 23 日 24 时开启调价窗口，汽油价格每吨上调 2000 元"

viewpoint 整体观点:
- 表达看法和判断
- 说明影响级别（全国性/行业性/局部）
- 说明影响方向（正面/负面/中性）
- 示例："这是全国性民生事件，直接影响数亿车主日常开支，推动新能源汽车消费"

causes 发生原因:
- 采用"主体 + 动机"结构
- 说明是谁做的、为什么做
- 示例："主体是国家发改委，动机是遵循既定的成品油价格形成机制，反映国际原油成本变化"

prediction 后续预测:
- 给出明确的方向判断
- 避免"可能...也可能"的两可表述
- 示例："预计油价将维持高位震荡，新能源汽车渗透率 2026 年底突破 50%"

advice 个体建议:
- 从投资/消费/就业角度给出可操作建议
- 示例："投资者关注新能源汽车产业链龙头；消费者购车应认真对比全生命周期成本"
```

## 阶段 4：建议生成提示词

**位置**: `_get_stage4_prompt()` (约第 178-203 行)

### 常见问题与修改建议

| 问题 | 症状 | 修改建议 |
|------|------|----------|
| 建议笼统 | "注意风险" | 要求"具体标的/方向" |
| 自_improvement 空 | 无内容 | 要求"技能/知识两个维度" |
| 机遇风险缺失 | 只说机遇 | 要求"机遇风险各 2 条" |

### 推荐 Prompt 模板

```python
suggestions 生成要求:

thinking 思维启发:
- 从新闻中提炼的认知升级
- 指出思维方式的转变
- 示例："从追求规模转向追求质量，从技术驱动转向商业价值驱动"

investment 投资建议:
- 给出具体的投资方向
- 区分"关注"和"规避"
- 示例："关注：新能源汽车产业链龙头、AI 基础设施提供商；规避：商业模式不清晰的纯概念公司"

self_improvement 个人提升:
- 技能层面：具体学什么
- 知识层面：深化哪个领域
- 示例："技能：学习提示词工程和 AI 工作流设计；知识：深化垂直领域专业知识"

opportunities_risks 机遇风险:
- 机遇：2-3 条具体机会
- 风险：2-3 条具体风险
- 格式：机遇 1、机遇 2...风险 1、风险 2...
```

5. 重新运行
   回到步骤 1
