## 关键风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| AI 输出 JSON 不稳定 | 解析失败率高 | 四层降级 + Prompt 调优 |
| Figma MCP 连接失败 | 无法获取设计稿 | 备选方案：手动导出截图 |
| 邮件客户端兼容性 | 样式错乱 | 使用 table 布局 + 内联 CSS |
| Token 超限 | API 调用失败 | 分批处理 + 摘要压缩 |
| 超时问题 | 阶段 2/3 超时 | 增加超时时间 + 重试机制 |
